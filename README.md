Smart Expense Tracker is a web-based application developed using Python and Flask. It allows users to manage their daily income and expenses in a simple and organized way. The application provides a clear summary of financial data and supports exporting records for further use.

This project demonstrates basic concepts of web development, database handling, and backend logic.

Features
Add income and expense transactions
View total income, total expense, and current balance
Filter transactions based on type and category
Delete unwanted transactions
Export all transaction data to a CSV file
Data stored using SQLite database

Technologies Used
Frontend: HTML, CSS
Backend: Python (Flask)
Database: SQLite
